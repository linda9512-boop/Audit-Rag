The criteria used to classify a product as a medical device under applicable regulatory jurisdictions for the Halcyon 4.0 (HAL 4.0) radiotherapy system are detailed as follows:

1. **United States**: In the United States, HAL 4.0 is classified as a Class II medical device under 21 CFR §892.5050, which corresponds to the product code IYE - Accelerator System, Linear [Regulatory / Regulatory Plan / D54978 HAL 4.0 Regulatory Plan Rev 5.0.pdf | Page 5, Section: 4. Type of Device].

2. **European Union**: HAL 4.0 is classified as Class IIb under the EU Medical Device Regulation 2017/745 (MDR), specifically applying Annex VIII Classification Criteria, Rule 9 [Regulatory / Regulatory Plan / D54978 HAL 4.0 Regulatory Plan Rev 5.0.pdf | Page 5, Section: 4. Type of Device].

3. **Canada**: In Canada, HAL 4.0 is classified as a Class III medical device according to the Medical Device Regulations (SOR/98-282), Schedule 1, Rule 8 [Regulatory / Regulatory Plan / D54978 HAL 4.0 Regulatory Plan Rev 5.0.pdf | Page 5, Section: 4. Type of Device].

4. **Japan**: HAL 4.0 is classified as a Class III device under the Pharmaceuticals and Medical Devices Act, Article 2, Paragraph 5 [Regulatory / Regulatory Plan / D54978 HAL 4.0 Regulatory Plan Rev 5.0.pdf | Page 5, Section: 4. Type of Device].

5. **China**: The device falls under Class III according to the National Medical Products Administration (NMPA) medical device classification category, specifically identified under 05-01-01 for medical electron accelerators [Regulatory / Regulatory Plan / D54978 HAL 4.0 Regulatory Plan Rev 5.0.pdf | Page 5, Section: 4. Type of Device].

The classification criteria involve assessing the device's risk, use, and regulatory requirements unique to each region. Each region has its framework for classifying medical devices based on their intended use and risk assessment.
