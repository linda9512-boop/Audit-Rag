Audit question: Is documentation readily retrievable for audit and regulatory submission?
Chunks used: 37
================================================================================
The audit question asks if documentation is readily retrievable for audit and regulatory submission. The provided excerpts indicate the following relevant points and evidence:

1. **Design History File (DHF) Location and Management**: 
   - The HAL 4.0 Design History File is organized and stored in the document control system Nucleus under a specific path, making it readily retrievable [Design Output Specification / D67731 HAL 4.0 Design History File Index Rev 05.docx.pdf | Page: 13-14].

2. **Regulatory Documentation and Storage**:
   - The HAL 4.0 MR1 Validation Plan states that validation and test results are documented in the Validation Summary Report and stored in the Device History File (DHF), supporting their ready retrieval [Design Validation / Verification and Validation Plans / D70753 HAL 4.0 MR1 Validation Plan Rev 02.docx.pdf | Page: 17-18].

3. **Archiving and Record Retention**:
   - Consistent record retention policies are in place as described in the Design and Development Plan (DDP) and the Quality Records Retention Schedule. This indicates that necessary records are maintained for easy access and retrieval [Regulatory / Test Reports - Regulatory Compliance / D104993953MIN-001 - HAL4.0 - IEC 60601-1.pdf | Page: 282-283].
   - There is mention of archiving the User Assistance Plan and related documentation, ensuring that finalized versions and significant changes are recorded and retrievable [Design Output Specification / Labeling / D65651 Halcyon-Ethos 4.0 IFU (P1058124-001-A) User Assistance Checklist Rev01.docx.pdf | Page: 6].

4. **Documentation Submission and Updates**:
   - Regulatory plans for electronic submission of documentation as they are approved indicate a systematic approach to ensuring documentation is current and retrievable [01. Design and Development Planning / 01. System / D30786 Halcyon 2.0 Regulatory Plan Rev 1.0.doc.pdf | Page: 50].

Overall, the evidence suggests that there are adequate systems and procedures in place to ensure documentation for the HAL 4.0 radiotherapy system is readily retrievable for audits and regulatory submissions.
