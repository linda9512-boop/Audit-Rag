Audit question: Are document control processes followed?
Chunks used: 37
================================================================================
The excerpts provide the following evidence regarding whether document control processes are followed for the HAL 4.0 radiotherapy system:

1. **Formal Document Control Processes in Place**:
   - Excerpts indicate that documentation related to the HAL 4.0, such as the Halcyon Evolve User Assistance Plan and various validation plans, include provisions for document approval and revision control through electronic signatures. These excerpts show that documents are reviewed and electronically signed by relevant personnel, thus ensuring control over document revisions and approvals [Design Transfer / Manufacturing Documentation / D60826-CHS-02_HAL4.0 Process Control Master Plan.pdf, Page 8-10], [Regulatory / Test Reports - Regulatory Compliance / D65933 HAL 4.0 IEC 60601-2-1 (2020) 201.10.101.5.4 - Accuracy of mechanical movements between CONTROL POINTS Report Rev 01.pdf, Page 1-2].

2. **Document Management Systems**:
   - The HAL 4.0 Design History File Index is located in a controlled document management system, as referenced in its Design History File (DHF). This system ensures that project documentation is properly stored and maintained [Design Output Specification / D67731 HAL 4.0 Design History File Index Rev 05.docx.pdf, Page 13-14].

3. **Procedure References**:
   - Multiple documents refer to DND01 Design and Development Procedure Rev 02, which outlines the phases of a project including documentation requirements, providing further evidence of a structured approach to documentation that supports the need for following document control processes [Regulatory / Test Reports - Regulatory Compliance / D104993953MIN-001 - HAL4.0 - IEC 60601-1.pdf, Page 292-293], [Regulatory / D70036 HAL 4.0 Software Description Report China Submission Rev 03 .docx.pdf, Page 23].

These excerpts confirm that Varian implements formal document control processes and procedures for the HAL 4.0 device, which are integral to maintaining rigor and compliance within their documentation management system.
